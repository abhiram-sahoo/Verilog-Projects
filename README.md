# 🔥 Verilog Projects Portfolio

## 📌 Overview
This repository contains a collection of **intermediate to advanced Verilog HDL projects**, covering core digital design concepts such as FSM, FIFO, memory structures, and control systems.

These projects demonstrate strong fundamentals in:
- Digital Logic Design
- RTL Design using Verilog
- Finite State Machines (FSM)
- Data Structures in Hardware (FIFO/LIFO)
- System-level design thinking

---

## 🧩 Projects Included

### 🚦 Traffic Light Controller Using FSM
- Designed using Finite State Machine
- Controls traffic signals based on timing logic
- Demonstrates sequential circuit design

---

### 🔄 Synchronous FIFO
- First-In-First-Out memory design
- Uses single clock
- Implements read/write pointer logic

---

### 🔁 Asynchronous FIFO
- FIFO with different read/write clocks
- Handles clock domain crossing
- Uses Gray code pointers

---

### 📦 Synchronous LIFO (Stack)
- Last-In-First-Out memory structure
- Push/Pop operations
- Stack pointer implementation

---

### 🎛️ PWM (Pulse Width Modulation)
- Generates PWM signals
- Used for motor speed / LED brightness control
- Duty cycle control implementation

---

### 🚗 Car Parking Management System
- FSM-based design
- Controls entry/exit logic
- Displays parking availability

---

### 🥤 Vending Machine
- FSM-based controller
- Accepts inputs (coins)
- Dispenses product based on logic

---
